## Section 2: Core AI Concepts

### 2.1 Understanding LLMs

**Learning Objective:** Participants understand what LLMs are and how they work.

* AI = **statistical predictor**, not a mind.
* Analogy: Autocomplete on steroidsâ€”trained on billions of text sequences.
* Key Takeaway: AI **mimics reasoning**, it does **not understand meaning**.

---

### 2.2 Tokenization

**Learning Objective:** Visualize how AI â€œseesâ€ language.

* Humans read words â†’ AI reads **tokens** (letters, words, subwords).
* Token sequences â†’ pattern predictions â†’ output.
* **Hallucination bridge:** Rare/unseen tokens can lead to plausible but incorrect outputs.

**Example Walkthrough:**

```
Prompt: "Chair squats are..."
Tokens: ["Chair", "squats", "are"]
Prediction: ["great", "safe", "for", "seniors"]
```

## Section 4: PRTC Framework

### 4.1 Why Prompts Matter

* Generic prompts â†’ generic results.
* Clear instructions â†’ **actionable outputs**.

---

### 4.2 PRTC Breakdown

| Element     | Description                                                                |
| ----------- | -------------------------------------------------------------------------- |
| Persona     | Role AI should take (e.g., physiotherapist)                                |
| Request     | Task to perform (e.g., list exercises, generate video)                     |
| Target      | Audience & tone (e.g., seniors, beginner-friendly)                         |
| Constraints | Boundaries: length, tools, style, safety (e.g., max 200 words, chair only) |

**Example Prompt (PRTC Style):**

> â€œTake `sarcopenia-guide.pdf` and create a 1.5-min explainer video for seniors. Narrate calmly, simplify technical terms, provide step-by-step chair/resistance band exercises, include nutrition tips, highlight safety, focus on beginner-friendly guidance, make it clear and concise.â€
